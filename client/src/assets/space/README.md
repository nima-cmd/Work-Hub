# Outside the canopy

Drop ONE image here to change what the cockpit window looks out on —
hyperspace streaks, a planet, a nebula, an approaching star destroyer.
Any `.png/.jpg/.jpeg/.webp/.svg`; first alphabetically wins.

No image = the built-in animated starfield (stars drift slowly, like cruising
at sublight). `client/src/data/spaceBackdrop.js` auto-discovers whatever you
drop; rebuild (`npm run client:build`) and the view changes.
